자바스크립트

웹브라우저에서 많이 사용하는 인터프리터 방식의 객체지향 프로그래밍 언어 - 자바는 컴파일 방식

자바스크립트 사용법

1. 태그에 직접 간단한 소스 작성해서 실행하는 방법 :인라인(inline) 방식
2. script태그에 작성하여 실행하는 방법 : 내부(internal) 방식
3. 별도의 js파일로 작성해서 가져다 사용하는 방법 : 외부(external) 방식
[js와 html 연결방식 3가지]
1)인라인(inline) 방식
2)내부(internal) 방식 : 같은 페이지
3)외부(external) 방식 : 외부 파일 호출

자바스크립트의 데이터 입출력

데이터 출력

다양한 방식으로 데이터를 "표시"

document.write() : 웹 상의 화면에 데이터 출력

alert() : 알림창 함수(경고상자)

consol.log() : 콘솔에 데이터출력 함수

innerHTML : 안에 엘리먼트값을 읽어와서 반환만 하는 기능

자바스크립트에서 태그 엘리먼트의 값을 읽거나 값을 변경할 때 innerHTML 속성 사용

데이터 입력

window.confirm()

어떤 질문에 대해 예/아니오 결과를 얻을 때 사용
확인 버튼과 취소 버튼이 나타나며 확인 버튼 클릭 시 true, 취소 버튼 클릭 시 false 반환

window.prompt()

텍스트 필드와 확인, 취소 버튼이 있는 대화상자
출력하고 입력한 메세지 내용을 리턴 값으로 돌려 받음

아이디로 접근하기

getElementById() : 반환타입 Element
해당 아이디의 요소를 선택함. id는 단일만이라 element이고 s로 안씀




태그명으로 접근하기

getElementsByTagName('태그명') : 반환타입 [] HTMLCollection

name으로 접근하기

getElementsByName() : 반환타입 Node list
