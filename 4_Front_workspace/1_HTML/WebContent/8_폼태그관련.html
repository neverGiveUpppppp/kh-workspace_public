폼 태그 관련

form태그 : 화면에서 사용자가 값을 입력할 수 있는 양식 제공 태그
　1)action속성 :

태그에 입력된 데이터들을 전송 받을 곳 지정 (ex:페이지,서버,클래스 등)
　2)method속성 : 정보를 보낼 방식을 적는 부분( get/post 방식 )
　　a.get : URL를 통해 정보를 전달 / 데이터 노출 o
　　b.post : HTTP Request Body를 통해 전달 / 데이터 노출 x
데이터 노출이 안되므로, id나 pw 같은 정보를 보내는데 쓰임



label 태그 폼 태그 안에서 정보를 입력 받는 태그가 어떤 역할을 하는지 알려주는 이름표 태그
서버로 정보 전달 x
화면 상 어떤 정보를 입력해야 하는지 알리는 목적으로 사용



legend 태그 : 필드셋에 대한 제목 작성



fieldset태그 : 필드셋으로 묶은 영역별로 구분




input 태그: 사용자에게 값을 받아오는 역할

text와 관련된 input 태그 속성값

type="hidden" : 화면에는 보이지 않는 상태로 데이터 전송

type="text"

문장 정보를 입력받을 때 사용

type="text"

type="password"

type="button", type="submit", type="reset"
